# LangChain Python 快速入门学习记录

这个仓库对应的是 LangChain 官方教程里 LangChain Python 的快速入门内容。
这里保留的是我学习时实际会看的 notebook、自己补的注释，以及少量为了本地运行做的适配，不是官方仓库的完整镜像。

## 仓库定位

- 主要内容在 `python/` 目录
- 以 `python/L1_fast_agent.ipynb` 到 `python/L9_HITL.ipynb` 这组 notebook 为主
- 在官方示例基础上补了中文注释、环境检查和本地 MCP 适配
- 不包含官方仓库的全部内容，也不维护 `js/` 相关部分

## 主要内容

- `python/L1_fast_agent.ipynb` 到 `python/L9_HITL.ipynb`
- `python/env_utils.py`
- `python/l5_local_mcp_server.py`
- `python/studio/` 下的 SQL agent 示例
